Manipulação de Sementes em Geradores Pseudoaleatórios

May 17, 2019 00:00 · 170 words · 1 minute read PNG Semestes Aleatórias

Você já usou funções como rnorm()? Se sim você já usou algum tipo de Gerador de Números Pseudoaleatório.

encher de coisas aqui

set.seed(1234)
n = 5000

amostra1 = rbinom(n= n,
                  size = 1,
                  prob = .5)

mean(amostra1)
## [1] 0.5014

Tivemos uma taxa de 0.5014com a semente \(1234\). Como funciona com outras sementes?

library(gganimate)

m = 10000000
n = 100
amostras = vector()

for(i in 1:m) {
  set.seed(i)
  
  amostra = rbinom(n = n,
                   size = 1,
                   prob = .5)
  
  amostras[i] = mean(amostra)
    
}


summary(amostras)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##    0.23    0.47    0.50    0.50    0.53    0.75
ngrupos = 20
grupo = rep(seq(1, ngrupos), times = m/ngrupos)
grupo = sort(grupo)

base = data.frame(dados = amostras,
                  grupo = grupo)

Note no entanto que escolhi um \(n\) pequeno com uma certa malícia. Por quê? Convergência assintótica.
