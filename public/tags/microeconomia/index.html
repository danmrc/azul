April 4, 2019

Dia desses eu concluí o primeiro release estável do Recon e inclusive já está disponível no CRAN para download, é só rodar install.packages(&quot;Recon&quot;) para instalar a última versão enviada ao repositório ou devtools::install_github(&quot;pedrocava/Recon&quot;) para baixar a versão mais recente. Com meu primeiro pacote finalmente no CRAN pensei fazer um post mostrando o que ele é capaz de fazer, afinal eu quero downloads. Ano passado eu fiz alguns posts aqui mostrando trabalho em progresso do pacote.

Usando dados da RAIS e Análise de Sobrevivência para entender desemprego
